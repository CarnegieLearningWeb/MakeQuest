Welcome to

At Globaloria we believe that making games is a fun way to learn. Designing educational games using real coding languages and professional design tools is exciting. Everyone can write real code – it’s fun even in your very first hour.

In the MakeQuest game, you edit JavaScript code to defeat "404" the evil villain. Along the way, you master computer science concepts like variables and functions.

Ready to go deeper? Access the full game code in the Globaloria Sandbox, remix and expand it to create your own game, then publish and share it with your friends!
